# sql-sales-insights-17-questions
This project showcases 17 real-time SQL business queries using a sales dataset, covering revenue analysis, customer insights, product performance, and profitability metrics.

# Overview
This project demonstrates my SQL skills by solving 17 real-time business problems using a sales dataset.
The analysis focuses on generating meaningful insights that help businesses make data-driven decisions.


# Objectives
Analyze revenue trends
Understand customer behavior
Evaluate product performance
Measure profitability and contribution
Solve real-world business questions using SQL
